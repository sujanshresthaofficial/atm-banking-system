# ATM Banking System

A simple console-based ATM Banking System developed using the C programming language. This project was created to practice core programming concepts such as loops, conditional statements, switch-case menus, and user input handling.

## Features

- Check account balance
- Deposit money
- Withdraw money
- Menu-driven interface
- Basic input validation

## Technologies Used

- C Programming Language
- Standard C Library (`stdio.h`)

## Concepts Practiced

- `do-while` loops
- `switch-case`
- `if-else` conditions
- User input handling with `scanf`
- Basic procedural programming

```bash
gcc ATMBankingSys.c -o atm
```

## How to Run

```bash
./atm
```
